  2. /metadata/key

    Permite obter alguns metadados. Chaves válidas são:
    countNúmero de entradas no dicionário
    first_wordPrimeira palavra do dicionário
    last_wordÚltima palavra do dicionário
    wordIdentificador da palavra do dia
